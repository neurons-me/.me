# API Reference

This section is the generated API surface for `.me`.

- Start with [ME Class](/api/classes/ME)
- Explore generated symbols in [API README](/api/README)

## What This Covers

The generated API documents the explicit class and type surface:

- `ME`
- `Memory`
- semantic path and operator types
- snapshot and replay structures

Low-level cryptographic helpers and key-wrapping internals are intentionally kept out of this public reference so the main API stays focused on the runtime surface most users actually touch.

The callable semantic DSL and `me["!"]` runtime mirror are documented separately in the guide pages, because they are proxy-driven runtime behavior rather than ordinary class methods.
